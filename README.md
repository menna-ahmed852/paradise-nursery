# Paradise Nursery

Welcome to the Paradise Nursery project! This is a React application built using Vite.

## Project Structure

```
paradise-nursery
├── index.html          # Main HTML file
├── package.json        # NPM configuration file
├── vite.config.js      # Vite configuration file
├── .gitignore          # Git ignore file
├── src                 # Source files
│   ├── main.jsx        # Entry point of the React application
│   ├── App.jsx         # Main App component
│   ├── index.css       # Global CSS styles
│   ├── components       # Reusable components
│   │   ├── Header.jsx   # Header component
│   │   └── Footer.jsx   # Footer component
│   ├── pages           # Application pages
│   │   └── Home.jsx     # Home page component
│   ├── hooks           # Custom hooks
│   │   └── useFetch.js  # Custom hook for data fetching
│   └── api             # API handling
│       └── client.js    # API client functions
├── public              # Public assets
│   └── robots.txt      # Robots.txt for web crawlers
└── README.md           # Project documentation
```

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd paradise-nursery
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to see the application in action.

## Features

- A responsive layout with a header and footer.
- A home page that serves as the main entry point.
- Custom hooks for data fetching.
- API client for handling requests.
